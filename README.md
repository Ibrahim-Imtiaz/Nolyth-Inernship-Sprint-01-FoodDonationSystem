# Food Waste Exchange API

A RESTful backend API built with FastAPI for connecting restaurants and organizations to reduce food waste.

## Features

- User Registration
- Secure Password Hashing (bcrypt)
- User Login
- JWT Authentication
- SQLite Database
- SQLAlchemy ORM
- FastAPI Swagger Documentation

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Passlib
- Python-JOSE
- Uvicorn

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Open:

http://127.0.0.1:8000/docs
